

### 1. 分析题意

*   **序列构成**：由 `0`, `1`, `2` 组成。
*   **操作规则**：
    1.  `2` 是“变色龙”：可以变成 `0` 或 `1`。
    2.  `00` 可消除：两个相邻的 `0` 可以直接删掉。
    3.  `11` 可消除：两个相邻的 `1` 可以直接删掉。
*   **目标**：求经过任意次操作后，序列能达到的**最短长度**。

---

### 2. 逐步解题思路

#### 第一步：化繁为简 —— 如果只有 0 和 1
假设没有 `2`，只有 `0` 和 `1`，规则是“相邻相同则消除”。
例如：`00110` -> `110` -> `0`。
这个规则直观上很难处理，因为消除哪一对 `00` 可能会影响后面。我们需要寻找一个**“不变量”**。

#### 第二步：核心大招 —— 下标奇偶取反（Parity Flip）
这是一个处理“相邻相同可消除”问题的常用技巧：**将所有偶数下标位置的数字取反（0变1，1变0）**。

*   **为什么要取反？**
    原规则是：消除相邻的 `00` 或 `11`。
    在取反后的序列中：
    *   原来的 `00`（下标 $i, i+1$）变成了 `01` 或 `10`。
    *   原来的 `11`（下标 $i, i+1$）变成了 `10` 或 `01`。
    
    **神奇的变化发生了：** 原本的“相邻相同可消除”，变成了新序列里的**“相邻不同可消除”**。

#### 第三步：数学不变量
在新序列中，规则变成了：**只要有相邻的 `0` 和 `1` 就能一起消掉。**
*   每消掉一对 `01` 或 `10`，`0` 的数量减 1，`1` 的数量也减 1。
*   **结论**：在新序列中，`0` 和 `1` 的数量差的绝对值 $|count(0) - count(1)|$ 是永远不会改变的。
*   只要序列中同时存在 `0` 和 `1`，就一定能找到相邻的 `01` 或 `10` 并消掉。最终，序列只会剩下一种数字（全是 `0` 或全是 `1`），剩下的长度正好就是数量之差。

#### 第四步：加入“变色龙” 2
现在考虑 `2`。在取反后的序列中，`2` 依然可以看作既能当 `0` 也能当 `1` 的占位符。
*   设取反后原有 $c_0$ 个 `0`，$c_1$ 个 `1`，$c_2$ 个 `2`。
*   我们需要把 $c_2$ 分配一部分给 `0`（设为 $x$ 个），剩下的给 `1`（$c_2 - x$ 个）。
*   最终长度为 $|(c_0 + x) - (c_1 + (c_2 - x))|$。
*   我们要枚举 $x \in [0, c_2]$，找到这个绝对值的最小值。

---

### 3. 代码实现逻辑讲解

结合代码看这三步：
1.  **预处理**：`for (int i = 2; i <= n; i += 2)` 循环里，把偶数位的 `0` 和 `1` 互换。这完成了“规则转换”。
2.  **计数**：统计转换后 `c0`, `c1`, `c2` 的个数。
3.  **贪心/枚举**：`for (int x = 0; x <= c2; x++)` 尝试所有可能的 `2` 的变法。
    *   其实不用枚举也可以：我们想让 $final0$ 和 $final1$ 尽可能接近。如果 $c_2$ 足够大，能抹平 $c_0$ 和 $c_1$ 的差距，最小值就是 0 或 1（取决于总长度的奇偶性）。

---

### 4. 算法复杂度证明

*   **时间复杂度**：$O(n)$
    *   遍历一次字符串进行取反：$O(n)$。
    *   遍历一次字符串统计数量：$O(n)$。
    *   枚举 $c_2$：$c_2 \le n$，所以是 $O(n)$。
    *   总时间：$O(n)$。
*   **空间复杂度**：$O(n)$
    *   需要存储长度为 $n$ 的字符串。如果不存字符串直接在读入时处理，可以优化到 $O(1)$，但 $O(n)$ 已足够优秀。

### 5. 核心逻辑伪代码

**Algorithm 1:** Calculate Minimum Difference  
**Input:** String $S$  
**Output:** Minimum absolute difference $\min\_len$

1.  $n \leftarrow \text{length}(S)$
2.  $c_0 \leftarrow 0, c_1 \leftarrow 0, c_2 \leftarrow 0$
3.  **for** $i \leftarrow 0$ **to** $n - 1$ **do**
4.  $\quad$ $char \leftarrow S[i]$
5.  $\quad$ $\triangleright$ *Perform logical negation for even indices*
6.  $\quad$ **if** $i \pmod 2 = 0$ **then**
7.  $\quad\quad$ **if** $char = \text{'0'}$ **then** $char \leftarrow \text{'1'}$
8.  $\quad\quad$ **else if** $char = \text{'1'}$ **then** $char \leftarrow \text{'0'}$
9.  $\quad$ **end if**
10. $\quad$ $\triangleright$ *Count the transformed characters*
11. $\quad$ **if** $char = \text{'0'}$ **then** $c_0 \leftarrow c_0 + 1$
12. $\quad$ **else if** $char = \text{'1'}$ **then** $c_1 \leftarrow c_1 + 1$
13. $\quad$ **else** $c_2 \leftarrow c_2 + 1$
14. **end for**
15. $\min\_len \leftarrow \infty$
16. $\triangleright$ *Assign $x$ variable characters to '0' and the remaining $c_2 - x$ to '1'*
17. **for** $x \leftarrow 0$ **to** $c_2$ **do**
18. $\quad$ $diff \leftarrow \left| (c_0 + x) - (c_1 + c_2 - x) \right|$
19. $\quad$ $\min\_len \leftarrow \min(\min\_len, diff)$
20. **end for**
21. **return** $\min\_len$

```cpp
#include <iostream>
#include <string>
#include <vector>
using namespace std;
int main()
{
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    string s;
    if (!(cin >> s))
        return 0;
    int n = s.size();
    // 第一步：偶数位置（1-based）0/1 取反，2 不动
    // s 是 0-based，所以 i = 1, 3, 5, ... 对应 2,4,6,... 位置
    for (int i = 1; i < n; i += 2)
    {
        if (s[i] == '0')
            s[i] = '1';
        else if (s[i] == '1')
            s[i] = '0';
        // s[i] == '2' 时不改
    }
    // 第二步：统计 0,1,2 个数
    long long cnt[3] = {0, 0, 0};
    for (char c : s)
    {
        if (c == '0')
            cnt[0]++;
        else if (c == '1')
            cnt[1]++;
        else if (c == '2')
            cnt[2]++;
    }
    // 第三步：枚举有多少个 2 变成 0，其余变成 1
    long long ans = (long long)1e18;
    for (long long i = 0; i <= cnt[2]; ++i)
    {
        long long c0 = cnt[0] + i;
        long long c1 = cnt[1] + (cnt[2] - i);
        long long cur = llabs(c0 - c1);
        if (cur < ans)
            ans = cur;
    }
    cout << ans << '\n';
    return 0;
}
```

```text
Function Solve(S):
    n = length(S)
    c0 = 0, c1 = 0, c2 = 0
    
    For i from 0 to n-1:
        char = S[i]
        # 对偶数下标进行逻辑取反
        If i is even:
            If char == '0': char = '1'
            Else if char == '1': char = '0'
        
        # 统计转换后的数量
        If char == '0': c0++
        Else if char == '1': c1++
        Else: c2++
    
    min_len = Infinity
    # 分配 c2 个变色龙，x 个变 0，c2-x 个变 1
    For x from 0 to c2:
        diff = absolute_value((c0 + x) - (c1 + c2 - x))
        min_len = min(min_len, diff)
        
    Return min_len
```

### 6. 复习补充知识（考试加分项）

1.  **奇偶性思想（Parity）**：
    这是解决“相邻消除”类题目最强的武器。通过对下标进行变换，将“静态”的数量统计转化为“动态”的消除平衡。

2.  **绝对值最小化技巧**：
    对于 $|(c_0 + x) - (c_1 + c_2 - x)|$，化简得 $|(c_0 - c_1 - c_2) + 2x|$。
    这是一个关于 $x$ 的等差数列。在考试中，如果你想更快，可以直接判断：
    *   如果 $c_2 \ge |c_0 - c_1|$，说明 `2` 的数量足够填平差距。最终长度就是 `(n % 2)`（总长度为奇数余 1，偶数余 0）。
    *   如果 $c_2 < |c_0 - c_1|$，说明 `2` 全部用来填补小的那个，还是填不平。长度就是 $|c_0 - c_1| - c_2$。

3.  **类似题目**：
    *   “只有 0 和 1，消除相邻不同位（01 或 10）” -> 直接求 $|count(0) - count(1)|$。
    *   “括号匹配” -> 也是一种消除，但有方向性（栈的应用）。本题没有方向性，所以更简单。

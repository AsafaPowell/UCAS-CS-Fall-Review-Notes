# UCAS CS Fall Review Notes
README 简介
本仓库整理了中国科学院大学计算机科学与技术学院秋季学期部分课程的复习资料，主要面向期末复习与知识梳理。

目前包含以下五门课程：
- 计算机网络
- 计算机系统结构
- 机器学习
- 高级人工智能
- 算法

仓库内容主要包括课程笔记、知识总结、重点整理、题型归纳及部分个人复习资料，供学习交流使用。
README 目录结构建议
## 仓库结构

- `Computer Networks/`：计算机网络
- `Computer Architecture/`：计算机系统结构
- `Machine Learning/`：机器学习
- `Advanced Artificial Intelligence/`：高级人工智能
- `Algorithms/`：算法
- `CLASS.md`：成绩组成
